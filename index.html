Autosave Transaksi

Berjalan di background otomatis mencatat transaksi anda.

Setiap anda transaksi aplikasi kami akan mencatan otomatis dan dapat mengetahui jenis transaksi nya berupa pemasukan atau pengeluaran,

namun dengan keterbatasan Aplikasi kami, kami masih belum bisa menghandle semua notifikasi, sehingga kamu bisa menggunakan fitur "tambah catatan keuangan" secara manual.

Manual add transaksi

Fitur cadangan untuk ketika aplikasi tidak dapat mencatat secara otomatis.

Anda bisa menambahkan data keuangan baik pengeluaran maupun pemasukan dan dalam 2 macam mode transaksi online atau offline,

contoh saja, aplikasi kami tidak dapat mencatat pengeluaran ketika anda melakukan transaksi offline seperti beli sesuatu di warung/toko-toko offline kemudian anda ingin tetap mencatatnya maka gunakanlah fitur ini.

Status user

Status anda Boros atau hemat.

Kami dapat mengetahui anda boros ataupun hemat dari perhitungan simple kedua tipe transaksi anda ( pengeluaran & pemasukan ), kami bandingkan keduanya sehingga akan memunculkan status anda hemat / boros.

Guest mode

simple to use without register or login user.

Anda tak perlu ribet membuat akun ataupun login, kami menyediakan fitur guest namun terdapat fitur change name dan profile,

anda dapat merubah nama dan profil pada menu profile yang telah kami sediakan.
